================================================================================
TRAFFIC VIOLATION DETECTION MODEL PACKAGE
================================================================================

This package contains everything you need to use the traffic violation detection
model in your own project.

PACKAGE CONTENTS:
-----------------
1. yolov8n.pt              - The pre-trained YOLOv8n model file (6.5 MB)
2. traffic_model.py        - Main model wrapper class with all detection methods
3. example_usage.py        - Complete usage examples for different scenarios
4. test_model.py           - Quick test script to verify installation
5. requirements.txt        - Python dependencies needed
6. README.md              - Full documentation
7. QUICK_START.md         - Quick start guide with code examples
8. .gitignore             - Git ignore file (if using version control)

INSTALLATION:
-------------
1. Copy this entire folder to your project
2. Install dependencies: pip install -r requirements.txt
3. Test installation: python test_model.py

QUICK USAGE:
------------
from traffic_model import TrafficViolationModel
import cv2

model = TrafficViolationModel()
image = cv2.imread('your_image.jpg')
detections = model.detect_objects(image)

for det in detections:
    print(f"{det['class_name']}: {det['confidence']:.2f}")
